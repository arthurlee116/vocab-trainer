# 英语词汇测验网站 (English Vocabulary Quiz)

一个交互式的英语词汇学习和测试网站,通过拖拽和手动输入两种方式帮助用户学习20个英语词汇。

## ✨ 功能特性

### 🎯 核心功能
- **词汇池展示**: 显示所有20个可拖拽的词汇单词
- **拖拽填空题**: 通过拖拽词汇到句子空位来完成填空
- **手动输入题**: 在输入框中手动输入正确的词汇
- **即时批改**: 提交后立即显示批改结果,正确为绿色,错误为红色
- **答案提示**: 错误答案会显示正确答案提示
- **分数统计**: 显示总分和各部分得分情况
- **重新测验**: 支持清空所有答案重新开始

### 🎨 交互特性
- 拖拽时有视觉反馈(透明度变化、边框高亮)
- 填写答案后输入框/空位样式变化
- 批改结果用颜色区分(绿色=正确,红色=错误)
- 悬停效果和过渡动画

## 📚 词汇列表

包含20个精选英语词汇:
- beacon, berserk, celestial, chasten, confiscate
- data, detract, encounter, epic, pantomime
- pessimist, precaution, prosecute, puncture, retaliate
- sham, uncouth, underscore, wholesome, wistful

每个词汇对应2道题目(1道拖拽题 + 1道输入题),共计40道题。

## 🎮 使用说明

### 第一步:查看词汇池
页面顶部的词汇池展示了所有可用的词汇单词。

### 第二步:完成拖拽题
1. 从词汇池中选择一个单词
2. 拖拽到 Part 1 题目中的空位(用下划线表示)
3. 释放鼠标完成填空
4. 可以重复拖拽同一个词汇到不同题目
5. 拖拽新词汇到已填写的空位会覆盖原答案

### 第三步:完成输入题
1. 在 Part 2 题目的输入框中手动输入词汇
2. 输入时输入框边框会变为橙色
3. 支持大小写混合输入(批改时会忽略大小写)

### 第四步:提交答案
1. 完成所有题目后,点击"提交答案"按钮
2. 系统会自动批改并显示结果
3. 正确答案显示为绿色背景
4. 错误答案显示为红色背景,并显示正确答案提示
5. 页面顶部会显示总分和详细统计

### 第五步:重新测验
点击"重新测验"按钮可清空所有答案,重新开始测验。

## 💡 答题技巧

- **大小写不敏感**: 输入"Beacon"或"beacon"都视为正确
- **空格处理**: 输入框中首尾的空格会被自动去除
- **词汇重用**: 同一个词汇可以用于多个题目
- **未作答处理**: 未填写的题目在批改时会标记为错误

## 🎯 核心组件说明

### App组件
- 管理全局状态(用户答案、批改结果、提交状态)
- 协调各子组件间的数据流
- 处理提交和重置逻辑

### VocabularyPool组件
- 展示可拖拽的词汇列表
- 实现拖拽源功能
- 提供拖拽视觉反馈

### DragQuizSection组件
- 渲染拖拽填空题
- 处理拖放事件(dragover, drop)
- 显示批改结果和正确答案

### InputQuizSection组件
- 渲染手动输入题
- 收集用户输入
- 显示批改结果和正确答案提示

## 📝 批改规则

1. **大小写忽略**: 用户答案和正确答案统一转为小写后比较
2. **空格处理**: 去除首尾空格后进行匹配
3. **完全匹配**: 必须完全匹配才视为正确
4. **空值处理**: 未填写的题目视为错误

## 🔄 状态管理

应用使用 React Hooks 进行状态管理:

- `dragAnswers`: 存储拖拽题答案
- `inputAnswers`: 存储输入题答案
- `submitted`: 标记是否已提交
- `dragResults`: 存储拖拽题批改结果
- `inputResults`: 存储输入题批改结果
- `scoreInfo`: 存储分数统计信息

## 🌟 设计亮点

1. **双模式学习**: 拖拽和输入两种方式,降低学习难度
2. **即时反馈**: 提交后立即看到批改结果
3. **视觉引导**: 丰富的视觉反馈帮助用户理解交互
4. **容错设计**: 大小写不敏感,用户体验友好
5. **可重复使用**: 支持重新测验,便于反复练习
